# Первые шаги {#getting-started}

## Установка {#installation}

### Требования {#prerequisites}

- [Node.js](https://nodejs.org/) версии 18 или выше.

::: code-group

```sh [npm]
$ npm add route-peek
```

```sh [pnpm]
$ pnpm add route-peek
```

```sh [yarn]
$ yarn add route-peek
```

```sh [bun]
$ bun add route-peek
```

:::
